# Testing-System
Система для тестирования студентов. 
Авторизация преподавателя или студента.
Возможности преподавателя:
Создание, редактирование, удаление, сохранение в бд mdf, открытие/закрытие тестов. Выбор типа вопроса: чекбокс/радиокнопка. Добавление, удаление вариантов ответов. Регулирование количества баллов за вопрос. Просмотр результатов прохождений теста вида: ФИ Группа Номер зачётки студента, количество набранных/возможное количество баллов, оценка в 10ти бальной системе.
